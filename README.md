# split-by-dates

Метод работает только с заранее отсортированным DataFrame

### Основные требования

- Возвращать DataFrame, разбитый по конкретной серии (колонке) на чанки желаемого размера
- Размер возвращаемых чанков должен быть **больше или равен** желаемому (`target_size`)
- Даты (или значения в указанной колонке) между чанками **не должны пересекаться**
- Последний чанк может быть любого размера
